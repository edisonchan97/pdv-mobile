# my-react-app
使用官方脚手架搭建的react+redux开发环境

一、装官方脚手架
```sh
npm install -g create-react-app
```
二、创建项目
```sh
create-react-app my-react-app
```
三、暴露配置文件
```sh
npm run eject
```
